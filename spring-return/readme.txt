>> Create Java Project.
>> Covert to Maven Project. 
	>>> Right Click on Project -> Configure -> Convert to Maven Project
	>>> pom.xml
>> Add Dependency/Library/Modules/Package to your project. 
	>>> follow the link :: https://mvnrepository.com/
	
	>>> Add dependency at proper tag. <dependencies>
	>>> spring-core
	>>> spring-context
	
	>>> Maven Dependecy folder will be added into your project classpath.
	
	
>> Create Main Class (Execution Point).
>> Create xml file. (beans.xml) at classpath
	>>> NOTE :: root tag attribute. 
	>>> ROOT TAG :: <beans>
	>>> Object declaration tag ::  <bean>
	
	
>> Load the spring container.
	>>> ApplicationContext.
	>>> getBean();
